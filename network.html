Operating System Terminology
TERM DEFINITION
Networking Classifications
LAN Local Area Network. Usually a collection of computers in a single building or complex.
MAN Metropolitan Area Network. A network of intermediate size, such as a local community.
WAN Wide Area Network. Links machines over greater distances.
Network Models
Bus Network A network topology where all of the machines are connected by a common communication line called a bus.
Star Network A network topology where one machine serves as a central focal point where all others are connected.
Protocols
CSMA/CD Ethernet Carrier Sense, Multiple Access with Collision Detection: A protocol used in a bus system where each machine monitors all messages, but keeps only those directed to itself. A machine waits until the bus is silent to transmit a message, and if another machine also transmits at the same time, they both wait for a random amount of time and then try again.
CSMA/CA Wifi Carrier Sense, Multiple Access with Collision Avoidance: A protocol used by WIFI to avoid collisions instead of simply detecting them.
